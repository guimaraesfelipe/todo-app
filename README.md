### Projeto em Java com Maven para planejamento de projetos e acompanhamento de suas tarefas.
Aplicação desenvolvida sobre o design patterns MVC (Model–view–controller), utilizando CRUD com banco MySQL para armazenamento dos dados da aplicação, Java Swing para desenvolvimento de interface gráfica e Junit para testes automatizados.
